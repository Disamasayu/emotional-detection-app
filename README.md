# emotional-detection-app

Proyek ini adalah aplikasi berbasis Python yang menggunakan **Watson NLP Library** untuk mendeteksi emosi dari teks. Aplikasi ini dikemas sebagai package dan dideploy menggunakan **Flask**.

## Fitur
* Mendeteksi emosi: Anger, Disgust, Fear, Joy, dan Sadness.
* Menentukan *Dominant Emotion*.
* Interface berbasis Web.
* Error handling untuk input kosong.
